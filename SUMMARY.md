1. Open: https://docs.google.com/spreadsheets/d/1OzdQ-B__cHqbuNp2GTRUxYqxP9t8F1d_SmPeOga1yDo/edit?usp=sharing
2. Go to Extensions > Apps Script
3. Copy code from GOOGLE_APPS_SCRIPT.js
4. Paste and save (Ctrl+S)
5. Deploy > New deployment > Web app
6. Set "Who has access" to **Anyone**
7. Copy the Web App URL
8. Update script.js: Replace GOOGLE_SCRIPT_URL with your URL

### Step 2: Deploy Website (2 minutes)
Choose one:

**Option A: GitHub Pages (Recommended)**
```bash
git init

git commit -m "AutoHub Solutions website"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
Then: Settings > Pages > Select main branch > Save

**Option B: Netlify (Easiest)**
- Go to netlify.com
- Drag and drop project folder
- Done!
